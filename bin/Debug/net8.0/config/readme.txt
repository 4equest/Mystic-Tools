[Settings]
SettingName = 設定名
Path = アクティブなウィンドウの実行ファイルのパス
PathRegex = Pathの正規表現を使うか true or false
Brightness = 明るさ 0 to 255
Mode = Static 下のColorに従う Video VideoPathの動画を再生
VideoPath = ModeがVideoの時はここにある動画をループ再生(キーボードがチャタリングを起こしたり反応しなくなったりする　反応しなくなったらUSB抜き差しで治るはず　でも壊れても知らん)
FrameRate = 10くらいがおすすめ　高くしすぎても20ちょいが限度なはず　高くするほどキーボードがフリーズする可能性が上がる
